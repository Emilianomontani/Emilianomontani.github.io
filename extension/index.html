Extensión para verificar errores básicos del criterio 1.3.1 Información y relaciones.

Esta aplicación está en fase beta. Puede contener fallos puntuales en los resultados lanzados. Se debe realizar una comprobación manual para verificar el contenido.

Se aplica un rol de estado para que la lista se anuncie a las personas usuarias que navegan con productos de apoyo. La extensión es funcional navegando con teclado e incluye un foco visible y un contraste de más de 4,5 en todos sus textos y elementos interactivos.

  • Verifica si hay párrafos vacíos o con " ".
  • Si hay etiquetas con <b>, <u>, <small>, <br>, <Font>.
  • si hay etiquetas <div> o <span> con texto incluido directamente. Se indica que ignore si están dentro de enlaces, botones, <li>, encabezados y etiquetas con el role="button" o role="link", pero puede contener fallos por la complejidad de ciertas estructuras HTML.
  • si hay etiquetas <i> y si ellas tienen el atributo aria-hidden="true".
  • si hay encabezados vacíos.
  • si hay encabezados vacíos dentro de tablas.
  • si hay más de un encabezado h1>.
  • si hay listas vacías <ul> o <ol>.
